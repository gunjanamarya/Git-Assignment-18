# Git-Assignment-18

1. Create a Github Account 
   
2. Create a public repository with name Git-Assignment-18

3. Clone the repository in your local machine `git clone https://github.com/gunjanamarya/Git-Assignment-18.git`
   
4. Run `cd Git-Assignment-18` 

5. Create a new branch : `git checkout -b middleware`

6. Copy `Backend` folder here and check the status of working directory with `git status`

7. Add folder to staging area : `git add Backend`

8. Commit the changes : `git commit -m "Add Backend folder to middleware branch"`

9. Push to middleware branch of remote repository : `git push origin middleware`

10. Checkout to master branch : `git checkout master`
    
10. Create a new branch : `git checkout -b ui`

11. Copy `Frontend` folder here and check the status of working directory with `git status`

12. Add folder to staging area : `git add Frontend`

13. Commit the changes : `git commit -m "Add Frontend folder to ui branch"`

14. Push to ui branch of remote repository : `git push origin ui`
    
15. Checkout to master branch : `git checkout master`
    
16. Add README : `git add README.md`
    
17. Commit the changes : `git commit -m "Add README"`
    
18. Push to the master branch of remote repository : `git push origin master`
